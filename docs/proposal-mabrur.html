Fitur Utama

Solusi Lengkap dalam
Satu Aplikasi

☯

Tuntunan Ibadah

Panduan langkah demi langkah untuk umrah (5 tahap) dan haji (6 tahap) lengkap dengan teks Arab, transliterasi Latin, dan terjemahan Indonesia.

◉

Geofence Miqat

Deteksi otomatis kedekatan dengan batas miqat menggunakan GPS. Peringatan ihram ketika jamaah mendekati zona miqat dalam radius 3 km.

⚠

SOS Darurat

Sistem darurat satu tombol untuk situasi medis, tersesat, atau kehilangan barang. Lokasi otomatis dikirim ke muthawwif rombongan.

☷

Doa & Bacaan

Koleksi doa haji & umrah dengan teks Arab, Latin, dan terjemahan. Siap digunakan kapan saja selama ibadah.

⏱

Jadwal Rombongan

Timeline perjalanan per rombongan dengan status waktu (akan datang, berlangsung, selesai) dan notifikasi pengingat.

☉

Monitoring Real-time

Dashboard muthawwif untuk memantau lokasi, status ihram, dan kondisi seluruh anggota rombongan secara real-time.

Mengapa Memilih Mabrur?

✓
Offline-First Data tersedia tanpa internet. Panduan ibadah, doa, dan jadwal tersimpan lokal di device.

Keamanan Berlapis Enkripsi AES-256-GCM untuk data sensitif. JWT dengan token rotation. Password bcrypt 12 rounds.
✓
Geofence Cerdas Deteksi otomatis 5 zona miqat + Masjidil Haram. Peringatan ihram berbasis lokasi real-time.
✓
SOS Satu Tombol Darurat medis, tersesat, atau kehilangan. Lokasi otomatis dikirim. Aksi cepat: Maps, WhatsApp, Telepon.
✓
Multi-Platform Satu codebase untuk Android & iOS. Admin panel berbasis web. Tidak perlu install khusus untuk admin.
✓
Audit Trail Lengkap Setiap perubahan data tercatat: siapa, kapan, apa yang berubah. Transparan dan akuntabel.
✓
Real-time Monitoring Muthawwif dapat memantau posisi dan status seluruh jamaah secara live dari dashboard mobile.
✓
Scalable & Maintainable Arsitektur modular dengan TypeScript. Database migration otomatis. Mudah di-scale horizontal.
✓
Push Notification Notifikasi real-time untuk SOS, pengingat jadwal, dan peringatan geofence. Bahkan di background.
✓
Bahasa Indonesia Native Seluruh antarmuka dalam Bahasa Indonesia. Siap untuk i18n (Arab, Inggris) di masa depan.
10
Keamanan

Keamanan Data
Tingkat Enterprise

Data jamaah dilindungi dengan standar keamanan terbaik di setiap lapisan sistem.

Enkripsi Data Sensitif

Nomor paspor dan catatan medis dienkripsi dengan AES-256-GCM di level aplikasi. Bahkan admin database tidak bisa membaca data mentah.

Autentikasi JWT

Access token 15 menit + refresh token 30 hari dengan rotasi otomatis. Token disimpan terenkripsi di SecureStore device.

Role-Based Access

Tiga level akses (jamaah, muthawwif, admin) dicek di setiap endpoint server. Tidak bisa di-bypass dari client.

Rate Limiting

Maksimal 10 percobaan login per 15 menit. Melindungi dari serangan brute-force password.

Security Headers

Helmet middleware: HSTS, X-Frame-Options, CSP, XSS Protection. HTTPS wajib di production.

Password Hashing

bcrypt dengan 12 rounds salt. Password tidak pernah disimpan dalam bentuk teks biasa.
